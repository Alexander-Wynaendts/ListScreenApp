Entourage: List Screening

About the Program

This tool analyzes companies based on their website using natural language processing to classify them as either SaaS, Service, or Hardware.

For SaaS companies, the tool will provide:

  • A one-sentence description
  • Client type (B2B or B2C)
  • Industry
  • Revenue model
  • Region of activity

Important: You must provide a .csv file containing at least two columns: "Name" and "Website URL". The "Website URL" column should contain valid URLs without any empty values. If these conditions are not met, the program will return the message: "An error occurred while processing the CSV file."

The CSV file name will be used as a tag in Affinity, so please provide a meaningful file name.

Note: The program runs live but can only process the same file for up to 1 hour. It cannot process more than 1,000 companies in one go. If your file exceeds this limit, please split it into smaller batches.
